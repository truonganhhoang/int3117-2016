# Bài tập 3

## Mô tả bài toán
 - Cho một số nguyên có ít nhất 5 chữ số.
   Trả về true nếu: tổng các chữ số là số chẵn và chia hết cho 3 hoặc 5.

 - Maven
 - Eclipse IDE
 - TestNG Plugin
