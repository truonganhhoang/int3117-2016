## Hàm cần test

'''
public int TimNamNhuan(int nam) {

		if ((nam % 400 == 0) || (nam % 100 != 0 && nam % 4 == 0)) {
			int x = nam % 19;
			if (x == 0 || x == 3 || x == 6 || x == 8 || x == 11 || x == 14 || x == 17) {

				return 0;
			} else
				return 1;
		} else
			return 2;
	}
'''

## Biểu thức Branch:

'((nam % 400 == 0) || (nam % 100 != 0 && nam % 4 == 0))'
